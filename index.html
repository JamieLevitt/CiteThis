Privacy Policy for the CiteThis Web Extension

Last updated: 21/4/2025

1. Introduction

This Privacy Policy explains how the CiteThis Twitter Extension (“Extension”) collects and processes data when you use it. By using our Extension, you agree to the collection and use practices described in this policy.

2. Data Collection and Use

  • Post Text Processing: When you view a post, the Extension extracts the post's text and sends it to our designated API endpoint solely for analysis.
  • Internal Processing: The post text is processed by Python functions running internally. All content is processed in memory, and no data is stored, saved, or sent to any external service during or after processing.
  • No Personal Data Storage: No personally identifiable information or additional user data is collected or stored beyond what is necessary for processing the post text. The processed results are returned to your browser and displayed without linking to your identity.

3. Data Security

  • Secure Processing: All data transmissions between your browser and our API use secure protocols (HTTPS).
  • Ephemeral Handling: The post text is held temporarily in memory during processing and is immediately discarded afterward. No logs or backups are maintained that include your post text.

4. Third-Party Disclosure

  • No Sharing with Third Parties: We do not share, sell, or transfer any post text or any processed data to third parties. All processing occurs within our internal systems.
  • API Endpoint: The API endpoint is operated solely for the purpose of providing the analysis feature and is secured to ensure that your data is not misused.

5. Data Retention

  • The Extension does not retain or save any post text or related data. All information is processed in real time and discarded immediately after the result is returned.

6. User Rights

  • Transparency: You have the right to understand what data is processed by the Extension.
  • No Data Stored: Since we do not store any of your post text or personal information, there is no data retained that can be accessed, modified, or deleted.

7. Changes to This Policy

We may update this Privacy Policy periodically to reflect changes in our practices or legal requirements. Any changes will be posted on this page with an updated effective date.
